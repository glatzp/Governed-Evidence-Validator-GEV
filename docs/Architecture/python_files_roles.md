# Python Files & System Roles Manifest

This document provides a complete manifest of all active Python files within the **Governed Evidence Validator (GEV)** codebase and defines their specific architectural role in the two-pass Q&A verification process.

## GEV Python Codebase Map

| File Path & Link | Layer / Directory | Primary Role in the Verification Process |
